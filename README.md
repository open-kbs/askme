# Personal AI Site — template

An open-source template for a chat-first professional portfolio.
Visitors chat with an AI version of you instead of reading a static
resume — it can check your calendar, book calls, and take messages.

## What you get

- **AI chat** — an LLM persona built from your career data, wired to
  real tools (calendar, bookings, contact form).
- **Calendar panel** — live free/busy from Google Calendar.
- **Two-phase booking** — visitor requests a slot → you get
  approve/reject links → they get a confirmation email.
- **Contact form** — rate-limited, honeypot-protected.

Calendar, bookings, and contact form are optional — enable them anytime
after your site is running.

## Getting started

Clone the repo and open it in your coding agent (Claude Code, Cursor,
Codex, or similar):

```
git clone <this-repo> my-site && cd my-site
```

Then say **"set up this project"**.

The agent walks you through everything: your profile, avatar, and
either local dev or deployment to <a href="https://openkbs.com" target="_blank">OpenKBS</a>. You
answer questions — the agent does the rest.
